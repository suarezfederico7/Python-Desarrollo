# Suarez Federico 3er Entrega

Contenido:
- Pagina de Inicio con botones de "Acerca de mi", "Crear Articulo", "Listar Articulos", "Login/Logout".
- Permisos sobre los botones para que sea usuario logueado.
- Formulario de Creacion de Articulos.
- Pagina para Listar los Articulos cargados con botones de detalle, modificacion y eliminacion.
- Mensaje de Creacion exitosa, no hay articulos disponibles.
- Botones para volver a la pagina principal.
- Perfil de Usuario para creacion o modificacion y manejo de Avatar.

Antes de empezar recuerde instalar los programas necesarios con el comando: pip install -r requirements.txt (recuerde de ejecutar este comando en el directorio donde se encuentre el archivo requirements.txt)
